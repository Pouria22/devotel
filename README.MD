## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/insurance-form-app.git

# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build the application
pnpm build

# Start production server
pnpm start
```
